Now iOS

像给自己发消息一样开始记录。Now 把快速捕捉和后续整理接在一起，让想法先被接住，再慢慢长成清晰内容。

  • 对话式记录，随手就能开始。
  • 支持文字与图片，记录速度更快。
  • 通过 iCloud 在 iPhone 与 Mac 间保持同步。

Now macOS

把快速记录带进简洁专注的 Markdown 工作空间，让片段继续长成完整内容。

  • 像发消息一样开始一条笔记。
  • 编辑 / 预览 / 分屏模式更适合长时间整理。
  • iCloud 同步与协作让内容在多设备间保持一致。
iOS
免费下载 Today

Today

不做复杂计划，只把想坚持的事放回今天。Today 会每天自动生成今天清单，让提醒、记录和热力图都围着今天展开。

  • 每天自动生成今天清单。
  • 提醒时间可自定义，不怕忘。
  • 连续记录和热力图让坚持看得见。
iOS
免费下载 记录Box

记录Box

少装几个 App，一次收住记账、笔记、时光和密码备忘。记录Box 把日常分散的小信息放进一个更稳定的盒子里。

  • 笔记、记账、记时光、密码备忘放在一起。
  • 支持标签、分类和预算等整理方式。
  • 数据可存本地，也可选择同步到自己的 iCloud。
