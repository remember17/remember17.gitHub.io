# 记录Box

`记录Box`是我个人开发的iOS APP

🎉  多功能工具箱：记账, 记时光, 记密码, 笔记, 搜索

**[AppStore下载记录Box](https://apps.apple.com/cn/app/id1579304692)**

<img src="../recordbox/api/recordbox.png" width = "200" height = "200" alt="下载记录Box二维码"/>

**💰  记账**
```objective-c
- 记录收入与支出
- 自定义账本
- 自定义类别
- 自定义类别背景图片
- 自定义每月起始日期
- 月预算，类别预算
- 记账页实时显示预算剩余
- 图表分析，汇总分析
- 饼图，柱状图，排序等
- 选择时间段过滤
```

**🗓  记时光**
```objective-c
- 记录时间和描述
- 自动区分当前记录的时间是未来或过去
- 实时计时，精确到秒
- 倒计时通知提醒
```

**🔐 记密码**
```objective-c
- 记录账号密码等内容，防止遗忘
- 可以生成随机密码
- 搜索快速查找
```

**📝  笔记**
```objective-c
- 记录文字与图片
- 每条笔记可以拍照或从相册添加多张图片
- 颜色标识，定义属于当前笔记的颜色
```

**🔍  搜索**
```objective-c
- 自定义搜索引擎
- 使用系统内置Safari浏览器
- 收藏常用关键词
```
