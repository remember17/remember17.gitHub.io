# 记录Box

`记录Box`是我个人开发的一款iOS APP

**欢迎使用：[App Store](https://apps.apple.com/cn/app/id1579304692)**

<img src="../recordbox/api/recordbox.png" width = "200" height = "200" alt="记录Box"/>

- 支持面容ID/触控ID/密码解锁
- 支持iCloud同步（iCloud云盘）

> 提示：使用iCloud同步，避免更换手机或卸载APP导致数据丢失：在iPhone系统`设置`中登录Apple账号，然后在iCloud设置中打开`iCloud云盘`和`记录Box`

**记账**
```objective-c
- 记录收入与支出
- 自定义类别
- 自定义每月起始日期
- 添加月预算，类别预算
- 图表分析，汇总分析
- 饼图，柱状图，排序等
- 选择时间段过滤
```

**记时光**
```objective-c
- 记录一个时间和主题
- 自动区分当前记录的时间是未来或过去
- 实时计时，精确到秒
- 倒计时通知提醒
```

**记密码**
```objective-c
- 记录账号密码等内容，防止遗忘
- 生成随机密码
- 搜索快速定位
```

**笔记**

```objective-c
- 记录文字与图片
- 每条笔记可以拍照或从相册添加多张图片
- 颜色标识，定义属于当前笔记的颜色
```
